# “Math” 方法

## $t.divide(...number)

对传入的数值求商（解决了浮点数精度问题）。

## 参数

<i style="color: #3492ff;font-weight: 700;">arguments (Number)</i>: 相除的数字。

## 返回

<i style="color: #3492ff;font-weight: 700;">(Number)</i>: 返回商数。

## 示例

```javascript

$t.divide(0.1, 0.3) // 返回 0.333333333333 而不是 0.33333333333333337

$t.divide(1, 2, 4, 8) // 0.015625

```
