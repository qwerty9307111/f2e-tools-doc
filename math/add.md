# “Math” 方法

## $t.add(...number)

对传入的数值求和（解决了浮点数精度问题）。

## 参数

<i style="color: #3492ff;font-weight: 700;">arguments (Number)</i>: 相加的数字。

## 返回

<i style="color: #3492ff;font-weight: 700;">(Number)</i>: 返回总和。

## 示例

```javascript

$t.add(0.1, 0.2) // 返回 0.3 而不是 0.30000000000000004

$t.add(1, 2, 3, 4) // 10

```
