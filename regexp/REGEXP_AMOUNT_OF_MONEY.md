# REGEXP

## $t.REGEXP_AMOUNT_OF_MONEY

匹配金额

```javascript
$t.REGEXP_AMOUNT_OF_MONEY.test('123.123')      // true

$t.REGEXP_AMOUNT_OF_MONEY.test('123,123.123')  // true

$t.REGEXP_AMOUNT_OF_MONEY.test('1,123.123')    // true

$t.REGEXP_AMOUNT_OF_MONEY.test('$123,123.123') // true

$t.REGEXP_AMOUNT_OF_MONEY.test('123,1.1')      // false
```
